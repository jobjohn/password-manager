# Password Manager

デスクトップで動作するローカル完結型のパスワード管理アプリです。

## 特徴

- **完全オフライン** - すべてのデータはあなたのPC内に保存され、外部サーバーには一切送信されません
- **強力な暗号化** - AES-256-GCM でデータを暗号化し、Argon2id でマスターパスワードから暗号鍵を導出します
- **自動ロック** - 一定時間操作がないと自動でロックされ、パスワード再入力が必要になります
- **トレイ常駐** - ウィンドウを閉じてもトレイに常駐し、すぐに呼び出せます
- **パスワード生成** - 文字種や長さを指定して安全なパスワードを生成できます
- **バックアップ** - 暗号化されたままエクスポート/インポートできます
- **マスターパスワード変更** - いつでもマスターパスワードを変更できます

## インストール

### 必要なもの

- [Rust](https://www.rust-lang.org/ja/tools/install)
- [Node.js](https://nodejs.org/ja) (v18 以上)
- [pnpm](https://pnpm.io/ja/installation)

### 起動方法

```bash
# 依存パッケージのインストール
pnpm install

# 開発モードで起動
pnpm tauri dev

# アプリのビルド
pnpm tauri build
```

## 使い方

### 1. Vault の作成

アプリを初めて起動すると、マスターパスワードの作成画面が表示されます。このパスワードがすべてのデータを暗号化する鍵になります。**絶対に忘れないでください。復元する手段はありません。**

### 2. エントリの追加

ロック解除後、「＋」ボタンから新しいエントリ（アカウント情報）を追加できます。

- **タイトル** - サービスの名前（例: Google, GitHub）
- **ユーザー名** - ログインID
- **パスワード** - パスワード（生成機能で自動作成も可能）
- **URL** - 対象サイトのURL（任意）
- **メモ** - 自由記入欄（任意）
- **タグ** - 分類用のタグ（任意）

### 3. 情報のコピー

エントリ詳細画面で、ユーザー名またはパスワードをクリップボードにコピーできます。

### 4. パスワード生成

エントリ編集画面の鍵アイコンからパスワード生成画面を開けます。
長さ（8〜64文字）と使用する文字種（小文字/大文字/数字/記号）を選択できます。

### 5. バックアップ

設定画面から、暗号化されたままVaultをファイルにエクスポートできます。
別のPCに移行したい場合やバックアップとして利用してください。

## データの保存場所

Vault ファイルは以下の場所に保存されます。

| OS | 保存先 |
|----|--------|
| macOS | `~/Library/Application Support/com.passwordmanager.app/vault.pwvault` |
| Windows | `C:\Users\<ユーザー名>\AppData\Roaming\com.passwordmanager.app\vault.pwvault` |
| Linux | `~/.local/share/com.passwordmanager.app/vault.pwvault` |

## セキュリティについて

- マスターパスワードは一切保存されません。正しいかどうかは復号を試みて判断します
- すべてのデータは AES-256-GCM で暗号化され、改ざん検知（認証タグ）も備えています
- Argon2id による鍵導出で、ブルートフォース攻撃（総当たり）への耐性を持たせています
- クリップボードにコピーしたパスワードは、一定時間経過後に自動で消去します
- ネットワーク通信は一切行わないため、外部への情報漏洩の心配はありません
