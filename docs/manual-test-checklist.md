# 手動テストチェックリスト

各マイルストーン完了時、およびリリースビルド前に `pnpm tauri dev` (または packaged app) 上で確認する。

## vault作成・ロック/アンロック (M3)

- [ ] 初回起動でCreateVaultView(vault作成画面)が表示される
- [ ] マスターパスワードでvault作成 → ロック解除済み状態(VaultListView)に遷移
- [ ] アプリを再起動 → UnlockViewが表示される
- [ ] 正しいパスワードでアンロック成功
- [ ] 誤ったパスワードでエラーメッセージが表示され、アンロックされない
- [ ] OS別のアプリデータディレクトリ(Tauriの`app_data_dir()`、macOSでは`~/Library/Application Support/<バンドル識別子>/`配下)に`vault.pwvault`が存在する

## エントリCRUD (M4)

- [ ] エントリを追加 → 一覧に表示される
- [ ] エントリ詳細を開く → タイトル/ユーザー名/パスワード/URL/メモ/タグが正しく表示される
- [ ] エントリを編集 → 変更が保存され、アプリ再起動後も反映されている
- [ ] エントリを削除(確認ダイアログあり) → 一覧から消え、再起動後も戻ってこない
- [ ] vaultファイルを手動で破損させて起動 → クラッシュせず「破損vault」相当のエラーになる

## パスワード生成器 (M5)

- [ ] 文字種チェックボックス(小文字/大文字/数字/記号)の組み合わせを変えて生成される文字が対応していることを確認
- [ ] 「紛らわしい文字を除外」ON時、l/I/1/O/0 が生成結果に含まれない
- [ ] 長さスライダーを動かすと生成パスワードの長さが追従する
- [ ] 「使用する」でエントリ編集フォームのパスワード欄に反映される

## 検索・タグフィルタ (M6)

- [ ] 検索ボックスにタイトル/ユーザー名/URLの一部を入力すると絞り込まれる
- [ ] タグチップをクリックすると該当タグのエントリのみ表示され、再クリックで解除される
- [ ] 検索語とタグを組み合わせた絞り込みが正しく動作する

## トレイ・自動ロック・クリップボード (M7)

- [ ] ウィンドウを閉じる(×ボタン)→ アプリは終了せずトレイに格納され、vaultがロックされる
- [ ] トレイアイコンのメニューから「表示/非表示」でウィンドウの表示切り替えができる
- [ ] トレイメニューの「今すぐロック」でvaultがロックされる
- [ ] トレイメニューの「終了」でプロセスが確実に終了する
- [ ] 一定時間(デフォルト5分)操作しないと自動的にロックされる
- [ ] エントリのパスワード/ユーザー名を「コピー」→ クリップボードに反映される
- [ ] コピー後25秒程度でクリップボードが自動的にクリアされる
- [ ] コピー直後に別の値を再度コピーした場合、古いタイマーでクリアされない(新しい値が保持される)
- [ ] 2つ目のインスタンスを起動しても新規ウィンドウは開かず、既存ウィンドウがフォーカスされる

## 設定・バックアップ (M8)

- [ ] マスターパスワード変更 → 変更後は新パスワードのみでアンロックできる(旧パスワードは失敗する)
- [ ] 常に最前面トグルの挙動が反映される
- [ ] テーマ設定の保存・再起動後の復元
- [ ] エクスポートで暗号化済みvaultファイルを任意の場所に保存できる
- [ ] インポートで別のvaultファイルに切り替えられ、アンロック画面に戻る
- [ ] ウィンドウの位置・サイズが再起動後も復元される

## ダークテーマ・自動起動 (フォローアップ)

- [ ] 設定画面でテーマを「ダーク」に切り替えると、ロック画面・一覧・詳細・編集・生成器・設定を含む全画面の配色が追従する
- [ ] テーマを「システムに合わせる」にした状態でOSの外観設定を切り替えると、アプリの配色も追従する
- [ ] 自動起動トグルをON → OS側の自動起動設定(macOSなら「ログイン項目」)に登録される
- [ ] 自動起動トグルをOFF → OS側の登録が解除される
- [ ] 自動起動でアプリが起動した場合でも、常にロック状態(UnlockView/CreateVaultView)から始まり、自動アンロックされない

## パッケージング

- [ ] `pnpm tauri build` がエラーなく完了する
- [ ] 生成された `.app` (macOS) を直接起動して上記の基本フローが動作する
